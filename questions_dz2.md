## ВОПРОС 3 - по сигнатуре трейта для отчёта

Задание требует вынести отчёт в трейт. Какой предпочтительный вариант сигнатуры:
trait Report {
    fn report(&self) -> String;  // возвращает строку
}

или

trait Report {
    fn print_report(&self);  // печатает напрямую
}
